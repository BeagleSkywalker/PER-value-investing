# Como instalar el indicador en Trading View
Mi recomendacion es verlo aqui : link a youtube

## Los pasos son simples:
1. Abrir desde Github el archivo # PE-Ratio_Capa_Investing.pine #.
2. Abrir Trading View, seleccionar cualquier compañia y abrir el gráfico de velas.
3. Abajo a la izquierda de la pantalla verás "Editor de Pine", hacer clic sobre esta opción.
4. En Github ya con el archivo abierto hacer clic en Copy Raw File , el ícono tiene dos cuadrados y esta a la derecha de la palabra Raw.
5. Una vez copiado, ir a Trading View, borrar todo el contenido del Editor de Pine y Pegar el contenido copiado.
6. Luego hacer clic en Guardar y Finalmente hacer clic en Añadir al gráfico.
7. Por ultimo, cerrar la pestaña del Editor de Pine
